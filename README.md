简介：
Android视图的字段和方法绑定，使用注释处理生成样板代码。
1、在字段上使用@BindView消除findViewById调用。
2、将列表或数组中的多个视图分组。 使用操作，设置器或属性一次操作所有这些操作。
3、通过使用@OnClick和其他方法注释方法来消除侦听器的匿名内部类。
4、通过在字段上使用资源注释来消除资源查找。
